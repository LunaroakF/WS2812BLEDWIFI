# WS2812BLEDWIFI
树莓派无线控制WS2812B灯带

# 接线  
请将灯带信号线(中间的那根)连接到树莓派18号口(靠外面从电源那端数第6个)，并给灯带通合适电源  

# WS2812BServer.py
这是树莓派上跑的程序

# 操作指南  
目前只弄了2560x1440屏幕的，将110珠灯带从屏幕右下角(屏幕面对人时候的右下角)开始顺时针贴(先长边后短边再长边最后短边)如图:
